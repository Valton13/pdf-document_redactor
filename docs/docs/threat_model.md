# Threat Model: Smart Document Redactor (STRIDE Analysis)

|-----------|-----------------|-----------------|--------|------------|--------------|

| **T04** | **I**nformation Disclosure | PII leakage via:<br>‚Ä¢ Metadata (XMP/Properties)<br>‚Ä¢ Hidden text streams<br>‚Ä¢ Temporary files | Critical (GDPR fine) | 1. Strip ALL metadata<br>2. Cryptographic redaction (`redact=True`)<br>3. tmpfs-only storage | Forensic analysis with `pdfid.py` + `exiftool` |
